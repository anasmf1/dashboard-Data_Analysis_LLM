# Projet LAMRANI – Dashboard & LLM Analysis

## Objectif
Développement d’un outil d’analyse et de visualisation des données commerciales
avec intégration d’un LLM pour l’aide à la décision.

## Structure du projet
- src/ : code source Python
- data/ : données brutes et traitées
- assets/ : interface et images
- reports/ : rapport et présentation
- outputs/ : résultats et dashboard BI
